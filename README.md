# Automated-Invigilator-Allotment-System
Teacher Allotment System (Desktop Application)

The Automated Invigilator Allotment System is a desktop-based application that helps colleges automatically assign teachers for examination invigilation duties.
It ensures fair duty distribution, reduces manual effort, and avoids scheduling conflicts.
